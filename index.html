== Description ==

This WordPress plugin Hide Admin Bar or Toolbar adding global options for hidding the frontend adminbar or toolbar according to logged in superadmins, administrators, users roles profiles. To disable the backend use instead Disable Admin Bar or Toolbar extension. To remove completely both frontend and backend Admin Bar or Toolbar use WP Admin Bar Removal or WP Toolbar Removal - Build 2016-01-05 - The configuration is Settable and this plugin is No Logo!

== Frequently Asked Questions ==

simply adds an options page to turn front end admin bar or toolbar off

How does it work?

The plugin add more global options to the settings menu to globally turn off (hide) the front end admin bar or toolbar and turn off (hide) the user personal options field in the profile to view it on dashboard.

= Can remove back end admin bar toolbar in WordPress 3.1+ or later? =

WordPress does not provide an easy method to remove the Admin Bar and Toolbar in the back end, so is possible use WP Admin Bar Removal or WP Toolbar Removal plugins that takes a different approach to override it!
